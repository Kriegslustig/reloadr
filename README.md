reloadr
=======

A WordPress-Plugin that reloads all pages when a file in the current theme is updated.

## Requirements
**Serverside**
* PHP 5.3
    - Anonymous functions

**Clientside**
* SSE (Server-Sent-Events)
[Further Information on SSE support](http://caniuse.com/#feat=eventsource)
